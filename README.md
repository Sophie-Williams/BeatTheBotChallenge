# BeatTheBot

BeatTheBot is a challenge where humans compete against a robot to play the [AE 3D Moto](http://www.windowsphone.com/en-us/store/app/ae-3d-motor/bb5f1317-735d-4e60-a100-9c3f1692ee7c) mobile phone game.

The point of [AE 3D Moto](http://www.windowsphone.com/en-us/store/app/ae-3d-motor/bb5f1317-735d-4e60-a100-9c3f1692ee7c) is to accumulate points by driving a moto as fast and as long as possible while quickly tilting the phone left and right to zig-zag through trafic.
